List Usage
Unorderd List An unordered list is used to present a list of items that not necessarily need to be shown in any specific order. To start this list you use the tag <ul>
Ordered List An ordered list is used to present a list of items that need to be shown in a specific order. To start this list you use the tag <ol>
Definition List A definition list is used when items come in pairs, e.g. glossary, vocabulary, questions and answers, etc. To start this list you use the tag <dl>
